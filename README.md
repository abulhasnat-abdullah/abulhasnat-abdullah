## Projects

**[Mars Rover Autonomy Stack](https://github.com/abulhasnat-abdullah/interplanetar_mars_rover)** &nbsp;·&nbsp; *ERC Remote*
SLAM + Nav2 + ROS2 architecture for competition-grade rover. Full Gazebo digital twin. Deployed for the European Rover Challenge.

**Warehouse AGV** &nbsp;`in progress`
Autonomous ground vehicle for logistics — Nav2 navigation, obstacle avoidance, kinodynamically-constrained path replanning in a simulated warehouse.

**[arm_visualizer](https://github.com/abulhasnat-abdullah/arm_visualizer)**
Real-time 3D robot arm visualization — joint states, end-effector trajectories, and workspace envelopes.

**[voice_controlled_robot](https://github.com/abulhasnat-abdullah/voice_controlled_robot)**
Speech-to-command interface mapping natural language to motor commands.
